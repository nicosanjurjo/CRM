 Este sistema de gestión de relación con los clientes (CRM) fue desarrollado a medida para una franquicia de Royal Prestige radicada en la ciudad de Villa Mercedes, San Luis.

 Se puede probar ejecutando el servidor local con Django, previa activación de un entorno virtual e instalación de las dependencias especificadas en "requirements.txt".
 
 Accediendo al directorio raíz, se podrá navegar por la interfaz de usuario y acceder a las distintas secciones del sistema.
En la sección "Clientes", se podrán dar de alta y modificar los clientes para asignarles actividades programadas,
las cuales podrán ser vistas, filtradas, modificadas y marcadas como completadas en la sección "Actividades".

 La base de datos presente solo contiene clientes ficticios para que se pueda probar el funcionamiento del sistema.
